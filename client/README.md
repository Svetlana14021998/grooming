<h1>Сайт груминг-салона</h1>
<h3>На данном сайте можно ознакомиться со всей информацией о работе груминг-салона, посмотреть результаты работ мастеров, ознакомиться с информацией о специалистах,
списком услуг, посмотреть и оставить отзыв, получить ответы на частые вопросы и прочитать полезные статьи. Также при можно заполнить форму обратной связи.</h3>
<hr/>
<h3>Страницы</h3>
<ol>
    <li>Главная странинца</li>
    <ul>
        <li>Перечень преимуществ</li>
        <li>Фотогалерея До-После</li>
        <li>Список популярных услуг</li>
    </ul>
    <li>Услуги</li>
    <ul>
        <li>Просмотр списка услуг с возможностью фильтрации (тип животного, цена, вид услуги)</li>
    </ul>
    <li>Мастера</li>
    <ul>
        <li>Карточки с информацией о специалистах</li>
        <li>Рейтинг и количество отзывов для каждого мастера</li>
    </ul>
    <li>Контакты</li>
    <ul>
        <li>Вся необходимая информация (адрес, телефон, соц.сети, схема проезда, часы работы)</li>
        <li>Форма для связи (можно оставить сообщение и оно придет на почту администратору)</li>
    </ul>
    <li>Отзывы</li>
    <ul>
        <li>Просмотр отзывов с фильтрацией (мастер, оценки, сотртировка по дате/оценке)</li>
        <li>Из каждого отзыва можно открыть карточку мастера</li>
        <li>Добавление отзыва</li>
    </ul>
  <li>Вопросы</li>
    <ul>
        <li>Можно ознакомиться с отвечами на часто встречающиеся вопросы</li>
    </ul>
  <li>Статьи</li>
    <ul>
        <li>Список статей с фильтрацией и пагинацией</li>
        <li>Можно перейти на страницу с содержимым статьи</li>
    </ul>
</ol>
